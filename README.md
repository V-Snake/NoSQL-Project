# NoSQL-Project

Pour lancer le projet:

## Back:

- ``cd backend``
- `` npm install``
- `` npm run start``

## Front:

- ``cd front``
- ``source venv/lib/activate``
- ``poetry install ``

### for Maps :

- ``poetry run streamlit run front/maps.py``

### for Authentification :

- ``poetry run streamlit run front/authentification.py``

### for  Agregations :

- ``poetry run streamlit run front/index.py``
